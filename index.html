Portfolio

Una pequeña muestra de algunos de los tantos clientes que han confiado en QIPA.

Omega Sports

Este proyecto es un sitio web ficticio.
Utilizamos colores acordes al video juego para el cual el equipo Omega e-Sports competía al momento de crear el proyecto.
El sitio cuenta con un carrusel de imágenes del juego, animaciones en Javascript, un acordeón animado en el cual se presentan los jugadores y sus roles.
Una tienda de productos relativos al merchandising de la organización y un formulario para contactar con ellos.
Tecnologías y librerías utilizadas:
- Diseño responsive - Bootstrap 5 - HTML5 - CSS3 - JavaScript - SASS - Google Fonts

Mi Bebé

Realizado para las madres primerisas en donde podrás adjuntar los datos de tu bebé y te ayuda con el seguimiento.
Además podrás chatear con otras madres que esten cerca tuyo.
"Una aplicación perfecta para vos y tu bebé"

Figma, photoshop e ilustrator

Veterinaria Pochettino

Sitio web para una veterinaria.
Cuenta con las distintas secciones a las que se dedican, salud y bienestar animal, alimentación, higiene, accesorios entre otros además cuenta con consumo de api de google maps y de formulario de contactar.

- Diseño responsive - Bootstrap 5 - HTML5 - CSS3 - SASS - Google Fonts

Silmar Bazar

Sitio web ficticio de un E-cormmerce de bazar.
Es un sitio basado en una mini empresa llamada silmar bazar en el cual se dedican a vender articulos de regaleria y bazar.El sitio cuenta con un slider de imagenes hecho en JavaScript, un formulario para que te lleguen las novedades y descuentos!

- Diseño responsive - HTML5 - CSS3 - JavaScript.
